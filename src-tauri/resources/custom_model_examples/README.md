# Custom Model Examples

YAML files in this folder describe how to load third-party ASR models in 「语墨」.

To use one, click **Settings → Models → Custom Models → Import Example** in the app, or copy the file manually to `~/.voiceink/custom_models/` and restart the app.

## Files

- `mimo.yaml` — Xiaomi MiMo V2.5 ASR (MLX INT4 quantized, ~8 GB on disk, Chinese / English)
